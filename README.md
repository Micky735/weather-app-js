# 🌤️ App Meteo Intuitiva (Open-Meteo API)

## Descrizione del Progetto
[cite_start]Questa applicazione consente di consultare le condizioni meteorologiche in tempo reale per qualsiasi città nel mondo[cite: 3]. [cite_start]Inserendo il nome di una città, l'app recupera le coordinate geografiche e visualizza dati aggiornati su temperatura, vento e condizioni generali[cite: 4, 6].

### Funzionalità principali:
* [cite_start]**Ricerca Geografica:** Converte i nomi delle città in coordinate tramite Geocoding API[cite: 4].
* [cite_start]**Dati in Tempo Reale:** Mostra temperatura (°C), velocità del vento e meteo (es. Sereno, Nuvoloso)[cite: 12, 13, 14].
* [cite_start]**Multi-Città:** Supporto per visualizzare il meteo di più località contemporaneamente[cite: 15].
* [cite_start]**Gestione Errori:** Notifica automatica se una città non viene trovata[cite: 16].
